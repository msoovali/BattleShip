# BattleShip
Battleship console/asp-net game in dotnet framework

**Database string can be changed in /WebApp/appsettings.json**

```
"ConnectionStrings": {
    "DefaultConnection":
    "server=localhost;database=battleship;user=root;password=toor"
  },
```
